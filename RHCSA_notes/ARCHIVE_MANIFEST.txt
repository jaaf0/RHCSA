TROUBLESHOOTING
---------------
Port 3000 already in use?
  PORT=3001 pnpm dev

Dependencies won't install?
  rm -rf node_modules pnpm-lock.yaml
  pnpm install

Build fails?
  pnpm check
  pnpm format
  pnpm build

Docker issues?
  docker system prune
  docker build --no-cache -f Dockerfile -t redhat-cheatsheet:latest .
